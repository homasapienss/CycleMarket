Эта панель используется для работы с остатками. Набор доступных действий зависит от роли текущего пользователя: менеджер работает только со своей точкой, а администратор может управлять остатками по всем магазинам и складам.

Ограничения роли

Эти правила можно использовать как ориентир для backend-проверок и навигации.

Менеджер работает только с остатками своей точки.
Администратор может управлять остатками любой точки без привязки к одному магазину.
Менеджер не должен менять свои учетные данные и привязку к магазину.
Создание и управление сотрудниками остается в зоне ответственности администратора.
