Добавление магазина или склада

Используй эту форму, чтобы зарегистрировать новую точку хранения или продажи, к которой потом можно будет привязывать сотрудников и остатки.

Точка успешно создана
После создания магазина или склада его можно будет использовать при назначении менеджеров.

Что ожидает backend

  • GET /admin/shops/new должен положить в модель пустой shopForm при необходимости.
  • POST /admin/shops/new должен принять как минимум shopName, shopType и address.
  • Опционально можно сохранить description, если оно есть в будущей модели точки.

Назначение формы

Эта страница нужна только администратору. Менеджер не должен создавать или редактировать точки.
